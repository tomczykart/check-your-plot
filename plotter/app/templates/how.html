How it's done

This website is designed with a python framework called Flask. It also uses Bootstrap for page layout and the Boxicons service for icons, Google fonts and Unsplash for images.

Search engine uses two separate services to gather information about the plot you searched for. The first one is called "ULKD" https://uldk.gugik.gov.pl/ for "usługa lokalizacji działek kastralnych" and it's an API for gathering plot information such as position of plot's borders coordinates.

The other service is a WMS https://integracja.gugik.gov.pl/cgi-bin/KrajowaIntegracjaEwidencjiGruntow service, which lets me download the crop of the map in the location you searched for.

For some reason, information about plot area in some regions of Poland is restricted, but information about plot geometry is not. This was the reason why I created the app in the first place - to let you quickly check the area, whether you are an architect, urban planner, developer, or simply just curious.
